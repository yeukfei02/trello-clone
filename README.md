# trello-clone

trello-clone

url: https://trello-clone-omega.vercel.app/

## Requirement:

- install yarn
- install node (v12+)

## Testing and run:

```
$ yarn

// development
$ yarn run dev

// production
$ yarn run production

// run type check
$ yarn run type-check

// use eslint and prettier to format code
$ yarn run lint
```

open localhost:5000
